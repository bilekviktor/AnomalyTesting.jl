Introduction

Comparison of maximum-likelihood and calibrated loss function for anomaly detection using SPTN model.

Anomaly detection terms:

  • fp - number of false positives etc.

  • p - positives = tp + fn

  • fpr - false positive rate = fp/p = fp/(tp + fn) = FALL OUT

  • tpr - true positive rate = tp/p = 1 - fnr = SENSITIVITY, RECALL, HIT RATE

  • tnr - true negative rate = tn/n = 1 - fpr = SPECIFITY, SELECTIVITY

  • fnr - ....................................... = MISS RATE

  • acc - ACCURACY = (tp + tn)/(p + n)

Model and loss comparion on anomaly detection

Comparsion of Right qunatile take functions with different percentage of train data around quantile. We will use 10000 iterations to train the models.

Right qunatile take - only qunatile

m1 = sptn(2, 9, 2)
x_train = flower2(100000, npetals=9)
x_val = flower2(1000, npetals=9)
x = RandomBatches(x_train, 1000, 10000)

qx = (right_quantile_take(m1, batch, 0.05, 0.00) for batch in x)

loss(x) = -mean(logpdf(m1, x))

ps = Flux.params(m1)

opt = ADAM(0.01)
Flux.Optimise.train!(x -> loss(getobs(x)), ps, qx, opt)

qval = right_quantile_take(m1, x_val, 0.05, 0.00)
println("Final right QT loss: ", loss(qval))

Final right QT loss: 4.02186473748809
